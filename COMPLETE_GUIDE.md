# 🎵 歌牌点歌系统 - 完整项目文档

## 项目概述

**歌牌点歌系统（Karuta Jukebox）** 是一个创新的歌牌游戏配套程序，集合了卡牌游戏机制与音乐播放功能。该系统支持 **Android 和 Windows 双端**，为大型地方赛事提供一体化解决方案。

## 核心功能

### 🎮 对局功能
- **卡组管理** - 支持多个独立卡组配置
- **随机抽卡** - 从卡组中随机生成对局卡牌
- **多歌映射** - 一张卡可对应多首歌曲，随机播放
- **对局判定** - 成功/失败状态管理和及时反馈
- **中场休息** - 定时插入休息音乐，支持手动继续

### 📊 数据管理
- **实时监控** - 后台面板实时显示卡组状态
- **误操作保护** - 支持手动恢复已出局卡牌
- **统计追踪** - 记录每回合成功/失败统计
- **状态导出** - 保存游戏进度数据

### 🎵 音乐功能
- **多格式支持** - MP3, WAV, FLAC, OGG, AAC
- **限时播放** - 每首歌自动限制在 10-30 秒
- **音量控制** - 实时调整播放音量
- **进度显示** - 播放进度实时反馈

## 配置说明

### 主配置文件 (`config/config.txt`)

| 参数 | 说明 | 示例 |
|------|------|------|
| `music_folder` | 音乐文件夹路径 | `config/music/` |
| `images_folder` | 图片文件夹路径 | `config/images/` |
| `min_duration` | 最小播放时长（秒） | `10` |
| `max_duration` | 最大播放时长（秒） | `30` |
| `rest_music` | 中场休息音乐文件 | `rest.mp3` |
| `failure_mode` | 失败处理模式 | `PASS` 或 `SKIP` |

### 卡组配置文件 (`config/decks/*.csv`)

```csv
image_name,work_name,songs
card1.png,作品A,song1.mp3|song2.mp3|song3.mp3
card2.png,作品B,song4.mp3
```

## 类设计详解

### Model 包

#### Song.java
```java
- fileName: String      // 音乐文件名
- displayName: String  // 显示名称
- fileFormat: String   // 文件格式
- duration: long       // 时长（毫秒）
- file: File          // 文件对象

Methods:
- fileExists()         // 检查文件是否存在
- getAbsolutePath()    // 获取绝对路径
```

#### Card.java
```java
- imageName: String              // 图片名称
- imageFile: File               // 图片文件
- workName: String              // 作品名称
- songs: List<Song>             // 歌曲列表
- isActive: boolean             // 是否在场上

Methods:
- addSong(Song)                 // 添加歌曲
- getRandomSong()               // 随机获取一首歌
- setActive(boolean)            // 设置活动状态
```

#### Deck.java
```java
- deckName: String              // 卡组名称
- cards: List<Card>             // 所有卡牌
- activeCards: List<Card>       // 在场卡牌

Methods:
- removeActiveCard(Card)        // 从场上移除卡牌
- addActiveCard(Card)           // 添加卡牌到场上
- getRandomActiveCard()         // 随机获取在场卡牌
- hasActiveCards()              // 检查是否有在场卡牌
```

#### GameState.java
```java
Enums:
- RoundState {IDLE, CARD_SELECTED, MUSIC_PLAYING, ...}
- Result {SUCCESS, FAILURE, PASS, SKIP}

Attributes:
- currentDeck: Deck             // 当前卡组
- currentCard: Card             // 当前卡牌
- currentSong: Song             // 当前歌曲
- roundState: RoundState        // 回合状态
- currentRound: int             // 当前回合号
- roundResults: List<Result>    // 历史结果
```

### Game 包

#### GameEngine.java
```
主要职责：控制游戏流程
- 管理回合生命周期
- 触发事件回调
- 协调音乐播放和结果提交

事件回调：
- onRoundStart()     // 回合开始
- onMusicStart()     // 音乐播放开始
- onMusicComplete()  // 音乐播放完成
- onRoundComplete()  // 回合结束
- onGameOver()       // 游戏结束
```

#### GameRules.java
```
主要职责：定义游戏规则
- 播放时长范围
- 中场休息间隔
- 失败处理模式
```

## 故障排查

### 启动失败
**症状**: 应用无法启动
**解决**:
1. 检查 Java 版本：`java -version`
2. 确保 JavaFX 库已正确安装
3. 查看控制台错误信息

### 无法加载卡组
**症状**: 出现 FileNotFound 异常
**解决**:
1. 检查 CSV 文件路径
2. 验证 CSV 文件编码为 UTF-8
3. 确保图片和音乐文件存在

### 音乐无声
**症状**: 游戏运行但无音乐播放
**解决**:
1. 检查音量设置
2. 验证音乐文件格式是否支持
3. 测试系统音量输出
